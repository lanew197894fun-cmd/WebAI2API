## [3.0.1] - 2025-12-16

### ✨ Added

- **故障转移系統**
  - 實現了基于 Pool 的自动故障转移：当某个 Worker 执行任务失敗（如 API 逾時、页面崩溃、被限流）时，系統会自动寻找下一个支援该模型的 Worker 进行重試。
  - **Merge 模式增强**：Merge Worker 内部也会在不同的配接器之间进行故障转移。

## [3.0.0] - 2025-12-14

### ✨ Added

  - 架构升级，支援同时管理多个瀏覽器实例和多个標籤页。
  - 實現了瀏覽器实例间的資料（Cookies/Storage）完全隔离。
- **Cookies 管理**
  - 新增 `/v1/cookies` 介面，支援取得指定 browser instance 的 Cookies。

### 🔄 Changed

- **配置系統重構**
